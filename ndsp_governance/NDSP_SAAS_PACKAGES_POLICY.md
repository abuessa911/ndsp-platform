# NDSP SaaS Packages Policy

## Status

Authoritative SaaS package policy.

## Active Packages

NDSP has four active packages:

1. Free
2. Pro
3. Elite
4. Institutional Suite

## Package Limits

### Free

Purpose: trial-level discovery and conversion driver.

- Markets: 1 market
- Assets: 2 assets
- Daily analyses: 1 analysis per day
- Visible named layers: none
- Decision depth: basic public output only
- Alerts: none
- API: no
- Webhooks: no
- Teams: no
- Reports: no

### Pro

Purpose: paid individual entry package with real value but clear upgrade pressure.

- Markets: 2 markets
- Assets: 20 assets
- Daily analyses: 15 analyses per day
- Visible named layers:
  - TDL
  - NMP
- Hidden by name:
  - Devil's Advocate
  - Nawaf Golden Alignment
- Their outputs may appear as general sanitized results without exposing their names.
- Alerts: basic limited alerts
- API: no
- Webhooks: no
- Teams: no
- Reports: limited or no advanced reports

### Elite

Purpose: full individual package for serious users.

- Markets: all supported markets
- Assets: 100 assets
- Daily analyses: 250 analyses per day
- Visible named layers:
  - TDL
  - NMP
  - Devil's Advocate
  - Nawaf Golden Alignment
- Alerts: advanced Telegram alerts
- Comparison: enabled
- Decision journal: enabled
- Scenario follow-up: enabled
- API: no by default
- Webhooks: no by default

### Institutional Suite

Purpose: commercial SaaS package for teams, companies, and institutional users.

- Markets: all supported markets
- Assets: 250+ or contract-based
- Daily analyses: contract-based
- Visible named layers:
  - TDL
  - NMP
  - Devil's Advocate
  - Nawaf Golden Alignment
- API Decision Output: enabled
- Webhooks: enabled
- Teams: enabled
- Reports: enabled
- Custom assets: available by contract
- Usage limits: contract-based

## Layer Visibility Rule

The 12 hidden internal layer names remain hidden in every package.

The four approved named layers appear depending on package:

- Free: none of the four names appear.
- Pro: TDL and NMP appear by name only.
- Elite: all four names appear.
- Institutional Suite: all four names appear.

## Output Rule

NDSP does not reduce decision value by hiding outputs.

NDSP hides internal names, formulas, weights, raw logic, source logic, and internal scoring.

Useful decision outputs remain available according to package depth.

## Final Rule

Free converts.  
Pro monetizes TDL and NMP.  
Elite unlocks the full named-layer experience for serious individuals.  
Institutional Suite turns NDSP into a commercial decision-support SaaS.
