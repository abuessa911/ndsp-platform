/* DSP design variables — paste into :root */
/* fonts: Reem Kufi · IBM Plex Sans Arabic · IBM Plex Mono · Sora */
:root{
  --void:#070A10; --ink:#0B0F17;
  --panel:rgba(18,24,36,.62);
  --panel-2:rgba(22,29,43,.55);
  --line:rgba(201,162,39,.18);
  --line-soft:rgba(255,255,255,.06);
  --gold:#D9B65A; --gold-bright:#F2D27A; --gold-deep:#9A7A22;
  --teal:#3FD3C6; --blue:#88A6FF; --rose:#E0607A;
  --txt:#EAEEF6; --txt-2:#9AA6BC; --txt-3:#5E6A82;
  --up:#56D6A0; --down:#E2657E; --warn:#E8B964;
  --radius:18px;
}
ملاحظات للمطوّر:
• الزر الذهبي = linear-gradient(135deg, --gold-bright, --gold-deep) ونصّه داكن #1a1303.
• البطاقات زجاجية: background:--panel + backdrop-filter:blur(12px) + حد --line-soft، الزوايا --radius.
• الاتجاه: استخدم خصائص منطقية (inset-inline، padding-inline) ليعمل القلب بين RTL/LTR تلقائيًا.
• الشعار: <g id="hunterSeal"> يُعرّف مرة ويُستدعى بـ <use href="#hunterSeal"/> في كل صفحة.
• الأرقام بخط Mono مع font-feature-settings:"tnum" لمحاذاة ثابتة.
