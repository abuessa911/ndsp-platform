# NDSP Trial And Activation Governance

Trial model:
- Total seats: 50
- Specialist/academic: 10
- Normal beginner: 25
- Premium invite-only: 15
- Trial duration: 16 days

Activation policy:
- Normal beginner: no admin activation, no invite required.
- Specialist/academic: admin activation required, no invite required.
- Premium invite-only: admin activation required and invite code required.

Canonical endpoints:
- /api/seats/status
- /api/trial/activation-policy
- /api/trial/invites/validate
- /api/trial/register/ordinary
- /api/trial/register/professional
- /api/trial/register/private-invite
