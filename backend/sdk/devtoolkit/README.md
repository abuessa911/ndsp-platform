# DEV-001 — NDSP Developer Toolkit

Official developer toolkit for NDSP.

## Commands

```bash
./backend/tools/ndsp create service CFG-001 configuration-center "Configuration Center"
./backend/tools/ndsp validate
./backend/tools/ndsp doctor
./backend/tools/ndsp registry
```

## Purpose

- Generate services using ENG-001.
- Validate service structure.
- Scan repository health.
- List service registry data.

## Rule

New services should be generated through this toolkit.
