# NDSP Architecture Specification v2.0

NDSP is an Intelligent Decision Operating System, not a trading bot and not a signal platform.

## Architectural Layers

1. Decision Intelligence Core
2. Decision Governance Core
3. Completed Decision Service
4. Platform Services
5. Client Applications

## Golden Rule

The official decision source is:

Completed Decision Service

No portal, bot, admin panel, Telegram service, mobile app, or future execution service may read directly from decision engines.
