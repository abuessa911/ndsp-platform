# NDSP Sovereign Command Interface Spec

Generated: 20260629_095927

## Direction

A premium institutional command-center interface for Nawaf Decision Support Platform.

## Design Principles

- Deep black background.
- Calm gold institutional accents.
- Large decision cards.
- Right-side command navigation.
- Colored decision radar with user-facing explanation.
- Beginner-first user experience with optional advanced details.
- Arabic RTL first.
- No noisy gaming-style dashboards.

## Radar Color Semantics

- Green: supportive environment / stable reading / no visible block.
- Gold: strong internal alignment / golden agreement.
- Yellow: monitoring / caution / decision not complete.
- Red: blocked / governance conflict / Devil's Advocate block.
- Blue: contextual check / neutral / insufficient data.
