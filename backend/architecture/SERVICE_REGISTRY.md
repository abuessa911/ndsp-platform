# NDSP Service Registry

## Platform Services

| ID | Service | Responsibility |
|---|---|---|
| SRV-001 | Auth Service | Login, session, account identity |
| SRV-002 | Admin Service | Administrative operations |
| SRV-003 | Trial Service | Trial registration and seats |
| SRV-004 | Payment Service | Payment and package activation |
| SRV-005 | Alerts Service | Alerts and notification policy |
| SRV-006 | Market Adapter Service | Live market data adapter |
| SRV-007 | Decision Package Service | Decision package delivery |
| SRV-008 | Decision Quality Service | Live decision quality |
| SRV-009 | Copilot Service | Explain completed decisions |
| SRV-010 | Telegram Service | Publish sanitized decision alerts |

## Rule

Services consume official decisions. They do not create official decisions.
