# NDSP Security Model

## Principles

- Public outputs must be sanitized.
- Hidden engine names remain hidden.
- Only approved public layer names may appear.
- No service exposes raw internal engine details.
- No public endpoint provides buy/sell instructions.
- Copilot cannot override governance.

## Decision Support Disclaimer

NDSP provides decision support only. It does not provide financial advice, buy/sell recommendations, or execution instructions.
