# NDSP Public Platform No Execution Language Policy

Generated: 20260629_095927

## Rule

The public NDSP decision-support platform must remain a decision-support interface only.

## Forbidden Language

The public platform must not use language that suggests:

- Trading execution
- Buy or sell orders
- Bot service
- Telegram private workflow
- Customer-facing execution channel

## Allowed Language

The public platform may use:

- Decision support
- Market reading
- Scenario reference levels
- Decision state
- Risk radar
- Monitoring
- Caution
- Blocked
- Supportive environment
- Governance
