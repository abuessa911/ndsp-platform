# DEV-001 — NDSP Developer Toolkit Contract

## Commands

- ndsp create service
- ndsp validate
- ndsp doctor
- ndsp registry

## Responsibilities

- Generate service skeletons using ENG-001.
- Validate service structure.
- Scan repository health.
- Display service registry.

## Non-responsibilities

- No deployment.
- No production service runtime.
- No decision logic.
